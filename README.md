# 📱 Apps Manager

To show the live demo <a href="https://apps-manager-eta.vercel.app/">Click Here</a>

A modern and user-friendly web application to manage and organize apps efficiently. This project allows users to track, manage, and interact with different applications through a clean and responsive UI.

## 🛠️ Technologies Used

- **Framework:** Next.js (App Router)
- **Library:** React.js
- **Programming Language:** JavaScript (ES6+)
- **Styling:** Tailwind CSS, daisyUI, CSS3
- **State Management:** React Context API
- **UI & Visualization:** Recharts
- **Routing & Navigation:** Next.js Routing System

## 🚀 Features

- Manage multiple apps
- View app details in a structured layout
- Clean and responsive UI design
- Fast performance with modern React architecture
- Mobile-friendly design
- Dynamic data handling using Context API
- Reusable and modular components
